ePuSta Elements — Live Demo

Demonstriert ePuStaInline und ePuStaGraph jeweils als JavaScript-Klasse und über data-Attribute.

⚠️ Diese Demo hat keinen echten ePuSta-Server. Alle Anfragen werden auf lokale Beispiel-JSON-Dateien umgeleitet (example/statistics-*.json), die mit echten Daten von mir_mods_00079200 (Juli 2025 – Juni 2026) befüllt sind.

ePuStaInline — JavaScript

Gesamtzahl der Zugriffe: 

const inline = new ePuStaInline(
  document.getElementById('inline-js'),
  'https://epusta.example.org',
  'oai:example.org:12345',
  '2022-01-01', '2024-12-31',
  'oas:content:counter'
);
inline.requestData();

ePuStaInline — data-Attribute

Gesamtzahl der Zugriffe: 

<span
  data-epustaelementtype="ePuStaInline"
  data-epustaproviderurl="https://epusta.example.org"
  data-epustaidentifier="oai:example.org:67890"
  data-epustafrom="2022-01-01"
  data-epustauntil="2024-12-31"
  data-epustatagquery="oas:content:counter"></span>

ePuStaGraph — JavaScript (mehrere Serien)

const graph = new ePuStaGraph(
  document.getElementById('graph-js'),
  'https://epusta.example.org',
  'oai:example.org:12345',
  'auto', '2024-12-31',
  [
    { label: 'Volltext', tagquery: 'oas:content:counter',          color: '#4e79a7' },
    { label: 'Abstract', tagquery: 'oas:content:counter_abstract', color: '#f28e2b' }
  ],
  'month'
);
graph.requestData();

ePuStaGraph — data-Attribute (eine Serie)

<div
  data-epustaelementtype="ePuStaGraph"
  data-epustaproviderurl="https://epusta.example.org"
  data-epustaidentifier="oai:example.org:67890"
  data-epustafrom="auto"
  data-epustauntil="2024-12-31"
  data-epustatagquery="oas:content:counter"
  data-epustagranularity="month"
  style="height:300px;"></div>
